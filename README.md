PitchSense is a modern, production-ready sports analytics and community dashboard built with **Next.js**, **Tailwind CSS**, and **Supabase**. It allows football fans to engage in live debates, provide tactical justifications for their match predictions, and compete against both their peers and a **Gemini AI** analyst to climb the global leaderboard.

---

## ✨ Features

- 🏟️ **Interactive Match Hub**: Explore upcoming major elite football fixtures, submit exact score predictions, and write tactical justifications.
- 🤖 **AI vs Human Analysis**: Compare your tactical breakdown directly against a masterclass tactical insight generated by Gemini AI.
- 💬 **Live Debate Feed**: A seamless, real-time social feed utilizing Supabase Realtime for instantaneous upvoting and threaded replies.
- 🏆 **Dynamic Leaderboard**: An automatically updating SQL-View tracking the best pundits based on a weighted scoring formula (Correct Winner vs Exact Score).
- ⚙️ **Fully Automated AI Backend**: Python scripts automated via GitHub Actions to flawlessly fetch live matches, generate AI insights, and score completed games daily using public ESPN data + Gemini AI.

---

## 🛠️ Tech Stack

- **Frontend**: Next.js 14 (App Router), React, Tailwind CSS, Lucide Icons
- **Backend / Database**: Supabase (PostgreSQL, Auth, Realtime)
- **AI / Automation**: Python, Google Gemini Flash-Lite, GitHub Actions
- **Data Source**: ESPN Public Scoreboard API (No API keys required!)

---

## 🚀 Architecture & Workflow

PitchSense runs completely autonomously in the background without needing manual intervention:

1. **`fetch_matches.py`**: Runs daily via GitHub Actions. It scrapes raw, public scoreboard data from ESPN and feeds it directly into Gemini's context window. Gemini acts as an intelligent parser to extract only the major elite upcoming fixtures (World Cup, Champions League, Premier League, etc.) and saves them to Supabase.
2. **`generate_ai_insights.py`**: Immediately after fetching matches, this script asks Gemini to generate a bold, highly technical tactical analysis and score prediction for each match.
3. **`update_results.py`**: Runs daily via GitHub Actions. It feeds ESPN data to Gemini to find the official full-time results of pending matches, securely scores all user predictions, and dynamically updates the global leaderboard.

> **Note:** The backend uses strict `application/json` MIME-types and context-feeding to completely bypass Google Search rate limits, making it 100% free and reliable.

## 💻 Local Development Setup

### 1. Prerequisites
- Node.js (v18+)
- Python (3.10+)
- A [Supabase](https://supabase.com) Account
- A [Google Gemini AI](https://aistudio.google.com/) API Key

### 2. Environment Variables
You will need to set up environment variables for both the frontend and the backend.

**Frontend (`web/.env.local`):**
```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

**Backend Scripts (`scripts/.env`):**
```env
SUPABASE_URL=your_supabase_url
SUPABASE_KEY=your_supabase_service_role_key
GEMINI_API_KEY=your_gemini_api_key
```

### 3. Running the Frontend
```bash
cd web
npm install
npm run dev
```
The app will be available at `http://localhost:3000`.

### 4. Running the Backend Scripts Manually
```bash
cd scripts
pip install -r requirements.txt
python fetch_matches.py
python generate_ai_insights.py
python update_results.py
```

## 🌍 Deployment Guide

To put PitchSense live on the internet so anyone can use it:

### Frontend (Vercel)
1. Push this entire repository to your GitHub account.
2. Go to [Vercel.com](https://vercel.com) and sign in with GitHub.
3. Click **Add New Project** and select your repository.
4. **Important**: Change the **Root Directory** to `web` (since the Next.js app is inside the `/web` folder).
5. Add your `NEXT_PUBLIC_SUPABASE_URL` and `NEXT_PUBLIC_SUPABASE_ANON_KEY` to the Environment Variables section.
6. Click **Deploy**.

### Backend Automation (GitHub Actions)
The repository is already equipped with a `.github/workflows/automation.yml` file. To activate it:
1. Go to your GitHub Repository -> **Settings** -> **Secrets and variables** -> **Actions**.
2. Click **New repository secret** and add:
   - `SUPABASE_URL`
   - `SUPABASE_KEY`
   - `GEMINI_API_KEY`
3. GitHub will now automatically fetch new matches every morning and score finished games every night!
